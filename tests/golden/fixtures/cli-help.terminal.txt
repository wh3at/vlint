$ vlint --help
Usage: vlint [options] [command]

Detect UI layout violations in declared targets.

Options:
  -V, --version    output the version number
  -h, --help       display help for command

Commands:
  check [options]  Run configured layout checks.
  browser          Manage the pinned Chromium browser.
  init             Create vlint.config.json.
  setup            Create config if needed and install the browser.
  help [command]   display help for command
